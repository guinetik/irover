# ChemCam: results ready while aiming

ChemCam stays in the same first-person mast view when a spectrum finishes processing. The toast still fires; **SPECTRUM READY** and **SEE RESULTS** appear on the ChemCam HUD bar so the player can open the experiment panel without any camera or mode change.

The instrument card overlay still has **SEE RESULTS** when ChemCam is selected but not activated.

If the player leaves ChemCam active view during a shot (e.g. **ESC**), the laser/integration sequence keeps running in the background. With ChemCam selected on the instrument card (**slot 2**), **FIRING…** / **INTEGRATING…** and a progress bar match the in-world HUD until the readout is ready, then **SEE RESULTS** appears as before.
